我的经历

  • 2015 年

    我接触到了 C++ 语言，但是由于当时年龄太小，并没有学懂。不过这是我首次接触到一个正经的编程语言（以前只有 Scratch）。

  • 2017 年

    初二，我接触到了 Python 语言。由于其语法简练，我经过一段时间的自学，逐渐掌握了 Python 语言的基础用法。当时，我自己尝试写了一个 Python module，叫“JingBh 模块”。它的功能非常简单，现在看起来似乎很幼稚，但这是我的第一个作品。

  • 2018 年

    我学习了 HTML 和 CSS 语言，制作了一个和 JingBh 模块一样简易（以及幼稚）的网站。后来，我发现了写网站的快乐，于是学习了 JavaScript、PHP 语言、MySQL 数据库等等。不过，当时我的水平很有限，还没有接触到各类框架，只知道这些语言最基础的用法。当时最令我骄傲的是，我制作了一个网盘系统。它实现了用户登录、文件存储（肯定要有）、预览和分享（参考百度网盘）。话说初二的我怎么这么闲...

  • 2019 年

    我接触并学习了 thinkPHP 框架，并在其基础上写了许多小应用（不过后来都废弃了）。

    我在这段时间内为了维护服务器，也学习了 Linux 系统的用法。

  • 2020 年

    我学习了 Laravel 框架，并通过 Laravel Mix 接触到了 Webpack，之后踩进前端大深坑，学习了 JavaScript 的各种用法（ES6/7、TypeScript）和各种框架（Vue、Nuxt），并且由于这个坑太深了，我仍在学习中。
